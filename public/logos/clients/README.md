# Logos dos clientes

Coloque aqui os logos das empresas que confiam na Cactus. O site usa estes arquivos com prioridade; se não existir arquivo local, tenta o Clearbit (quando disponível) e, por fim, exibe o nome em texto.

O Clearbit (logo.clearbit.com) pode estar indisponível ou encerrado. **Recomendado: adicionar os logos manualmente.**

**Nomes dos arquivos** (um por cliente, use o `slug`; PNG ou SVG):

| Slug            | Arquivo              | Empresa        |
|-----------------|----------------------|----------------|
| banco-bv        | `banco-bv.png`       | Banco BV       |
| eduzz           | `eduzz.png`          | Eduzz          |
| libert          | `libert.png`         | Libert         |
| usp             | `usp.png`            | USP            |
| beanalityc      | `beanalityc.png`     | Beanalityc     |
| zummit          | `zummit.png`         | Zummit         |
| arcelormittal   | `arcelormittal.png`  | ArcelorMittal  |
| havan           | `havan.png`          | Havan          |

**Onde conseguir os logos**

| Empresa       | Sugestão |
|---------------|----------|
| **Banco BV**  | Site oficial bv.com.br (rodapé) ou [Wikimedia Commons](https://commons.wikimedia.org/wiki/File:Logo_Banco_BV.png) (PNG/SVG). SeekLogo e LogosPNG também têm versões em PNG. |
| **Eduzz**     | Site eduzz.com (rodapé ou área de imprensa) ou busque “Eduzz logo png”. |
| **Libert**    | Site libert.com.br (rodapé) ou busque “Libert logo png”. |
| **USP**       | Site oficial de identidade visual: [scs.usp.br/identidadevisual/logotipos/](https://scs.usp.br/identidadevisual/logotipos/) — PNG transparente e vetor. Respeitar uso institucional. |
| **Beanalityc**| Site beanalityc.com (rodapé) ou busque “Beanalityc logo png”. |
| **Zummit**    | Site zummit.com.br (rodapé) ou busque “Zummit logo png”. |
| **ArcelorMittal** | [Wikimedia Commons](https://commons.wikimedia.org/wiki/File:Arcelormittal-logo.png), logotyp.us ou companieslogo.com (PNG/SVG). |
| **Havan**     | Site havan.com.br (rodapé/cabeçalho) ou busque “Havan logo png”. BrandLogos.org tem versão atual. |

Regra geral: prefira o site oficial ou área de imprensa; use PNG com fundo transparente quando possível. Respeite marcas e uso permitido.

Se existir `.png` e `.svg` para o mesmo slug, o site tenta o PNG primeiro.
